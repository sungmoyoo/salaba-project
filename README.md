# 개요
---
사라바 🧳
이 서비스는 '한달살기'를 주제로, 사용자가 숙박시설을 제공하거나 이용할 수 있고, 해당 지역의 행사와 소식뿐만 아니라 지역 커뮤니티를 형성할 수 있도록 한 플랫폼입니다. 여행과 커뮤니티 활동을 하나의 공간에서 간편하게 즐길 수 있도록 하는 취지에서 만들게 된 서비스입니다.

# 개발환경
---
- IntelliJ IDEA 2023.3.2 (Community Edition)
- JDK 21
- Gradle 8.5
- Spring boot 2.7.18
